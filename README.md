# tfg-agentes-ia
Repositorio del TFG "Diseño y desarrollo de agentes IA experimentales". Incluye prototipos, pruebas, experimentos, código de agentes, documentación técnica y resultados.
## Descripción
Este proyecto explora la implementación de arquitecturas cognitivas y sistemas multi-agente, evaluando su capacidad para resolver tareas complejas de forma autónoma.
